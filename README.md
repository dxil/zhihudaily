## Dependencies

```javascript
"dependencies": {
  "babel-runtime": "^6.9.2",
  "fastclick": "^1.0.6",
```

## Build Setup

```bash
# install dependencies
npm install

# server with hot reload at localhost:8000
npm run dev

#build for production with minification
npm run build
```
